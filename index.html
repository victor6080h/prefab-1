주요 안내 사항

📋 시스템 사용 안내
  • 각 페이지에서 입력한 데이터는 자동으로 브라우저에 저장됩니다
  • 페이지 이동 시 데이터가 유실되지 않으니 안심하고 사용하세요
  • 모든 점검 항목을 완료한 후 종합 대시보드를 확인하세요
⚠️ 주의 사항
  • 브라우저 캐시를 삭제하면 저장된 데이터가 사라질 수 있습니다
  • 정기적으로 데이터를 내보내기하여 백업하는 것을 권장합니다
  • 임계값을 초과하는 측정값이 있을 경우 즉시 보고하세요
✅ 품질 기준
  • 가속도: ±0.5g 이내
  • 기울기: ±2° 이내
  • 변형률: ±100με 이내
  • 제작 오차: ±5mm 이내
