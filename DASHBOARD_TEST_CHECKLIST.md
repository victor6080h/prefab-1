## 🔍 LocalStorage 검증

### Application 탭에서 확인
1. F12 → Application 탭
2. Local Storage → https://victor6080h.github.io 선택

**확인할 키:**
- [ ] `ACTIVE_REPORT_ID` 존재 확인
- [ ] `prefab_qms_report_XXX_qms_project_info` 존재
- [ ] `prefab_qms_report_XXX_qms_survey_data` 존재
- [ ] `prefab_qms_report_XXX_qms_acceleration_data` 존재
- [ ] `prefab_qms_report_XXX_qms_tilt_data` 존재
- [ ] `prefab_qms_report_XXX_qms_strain_data` 존재
- [ ] `prefab_qms_report_XXX_qms_fabrication_error` 존재

**데이터 구조 확인:**
```javascript
// 가속도 데이터 예시

  "maxTilt": "6.5",
```
